# RoboFriends

A fun React app that lets you search and filter a list of robots by name.  
Built with **Create React App**, fetches robot data from an API, and uses Robohash for cool robot avatars.

## Features
- Search bar to filter robots in real-time  
- Responsive design (works on mobile & desktop)  
- Clean UI with card layout  
- Loading state & error handling (if API fails)

## How to Run Locally
1. Clone the repo  
   ```bash
   git clone https://github.com/Huzefa077/robofriends_react.git
   ```

2. Go into the project folder  
   ```bash
   cd robofriends_react
   ```

3. Install dependencies  
   ```bash
   npm install
   ```

4. Start the development server  
   ```bash
   npm start
   ```
   ‚Üí Opens at http://localhost:3000

## Build for Production
```bash
npm run build
```
