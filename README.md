# 🤖 RoboFriends

A fun and interactive React application that displays robot profiles fetched from an API and lets users search through them in real time.

Built while learning React using the **Zero To Mastery – RoboFriends** mini-project.

## 🧠 What This Project Does

* Fetches robot data from an external API
* Displays each robot as a card
* Lets the user **search robots instantly**
* Updates the UI in real time as you type

This project is designed to demonstrate:

* Component-based architecture
* State and props
* API data handling
* React rendering flow

## 🧩 How It Works

1. The app fetches user data from an API
2. Each user is converted into a **robot card**
3. Typing in the search box filters robots
4. React re-renders only what changed

This shows how **state → UI → re-render** works in React.

## ⚙️ How To Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/robofriends.git

# 2. Go into the project
cd robofriends

# 3. Install dependencies
npm install

# 4. Start the app
npm start
```

Then open:

```
http://localhost:3000
```
